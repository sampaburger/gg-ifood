# GGK Analytics - Resumo iFood

App Streamlit para upload de 3 relatórios do iFood:

1. Relatório de Pedidos
2. Financeiro / Conciliação
3. Desempenho / Vendas

## Rodar localmente

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Publicar no Streamlit Cloud

Suba `app.py`, `requirements.txt`, `runtime.txt` e `README.md` na raiz do repositório GitHub e faça o deploy apontando para `app.py`.
